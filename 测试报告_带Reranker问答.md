## 输出的块内容

每个查询返回**5个精排后的文档**，前3个会用于生成答案。

**输出信息包含**:
- ✓ 文档标题/文件名
- ✓ 文档大小（字符数）
- ✓ 内容预览（前100字符）
- ✓ 完整内容（用于LLM）

**示例**:
```
[1] 关于印发济南市2025年家电以旧换新补贴实施细则的通知.md
    大小: 4399 字符
    预览: # 济南市商务局 济南市发展和改革委员会文件...
```
