## 2. 请求处理流程（详细版）

```
用户查询: "济南市家电补贴政策是什么？"
    │
    ↓
┌────────────────────────────────────────────────────────┐
│ 1. 安全网关 (Security Gateway)                          │
│    ├─ 验证JWT令牌                                       │
│    ├─ 检查用户权限 (RBAC)                              │
│    ├─ 速率限制检查 (Token Bucket)                      │
│    ├─ 输入验证 (长度、注入攻击、XSS)                   │
│    ├─ 内容过滤 (敏感词)                                │
│    └─ 审计日志记录                                     │
└────────────────┬───────────────────────────────────────┘
                 │ ✓ 验证通过
                 ↓
┌────────────────────────────────────────────────────────┐
│ 2. 会话管理 (Session Manager)                           │
│    ├─ 获取/创建会话                                     │
│    ├─ 加载历史对话 (最近5轮)                           │
│    ├─ 提取上下文实体                                   │
│    └─ 查询增强 (基于历史)                              │
└────────────────┬───────────────────────────────────────┘
                 │
                 ↓
┌────────────────────────────────────────────────────────┐
│ 3. 意图识别 (Intent Router)                             │
│    ├─ LLM意图分类                                       │
│    │   └─ 意图: policy_inquiry                         │
│    │       子意图: eligibility                          │
│    ├─ 实体提取                                          │
│    │   └─ 政策类型: 家电补贴                            │
│    │       地区: 济南市                                 │
│    ├─ 查询复杂度评估                                    │
│    │   └─ 复杂度: 0.4 (中等)                           │
│    └─ 选择处理链                                        │
│        └─ kb_chain (优先级1)                           │
└────────────────┬───────────────────────────────────────┘
                 │
                 ↓
┌────────────────────────────────────────────────────────┐
│ 4. KB检索链 (KB Chain)                                  │
│                                                          │
│    ┌──────────────────────────────────────────────┐   │
│    │ 4.1 向量检索 (Vector Retriever)               │   │
│    │     ├─ 查询向量化                             │   │
│    │     ├─ Milvus相似度搜索                       │   │
│    │     ├─ Top-K: 10, 阈值: 0.7                  │   │
│    │     └─ 返回5个相关文档                        │   │
│    └──────────────┬───────────────────────────────┘   │
│                   ↓                                     │
│    ┌──────────────────────────────────────────────┐   │
│    │ 4.2 政策分析 (Policy Analyzer) - 并行        │   │
│    │     ├─ 文档1: 分析中...                       │   │
│    │     ├─ 文档2: 分析中...                       │   │
│    │     ├─ 文档3: 分析中...                       │   │
│    │     ├─ 文档4: 分析中...                       │   │
│    │     ├─ 文档5: 分析中...                       │   │
│    │     └─ 提取结构化信息:                        │   │
│    │         • 申请条件                            │   │
│    │         • 补贴标准                            │   │
│    │         • 申请流程                            │   │
│    │         • 有效期限                            │   │
│    └──────────────┬───────────────────────────────┘   │
│                   ↓                                     │
│    ┌──────────────────────────────────────────────┐   │
│    │ 4.3 答案生成 (Answer Generator)               │   │
│    │     ├─ 基于分析结果生成答案                   │   │
│    │     ├─ 组织答案结构                           │   │
│    │     ├─ 添加引用来源                           │   │
│    │     ├─ 计算置信度: 0.85                       │   │
│    │     └─ 生成后续建议                           │   │
│    └──────────────┬───────────────────────────────┘   │
│                   ↓                                     │
│    ┌──────────────────────────────────────────────┐   │
│    │ 4.4 答案验证 (Answer Verifier)                │   │
│    │     ├─ 事实核查                               │   │
│    │     ├─ 一致性检查                             │   │
│    │     ├─ 完整性评估                             │   │
│    │     └─ 验证通过 ✓                             │   │
│    └──────────────┬───────────────────────────────┘   │
└───────────────────┼────────────────────────────────────┘
                    │
                    ↓
┌────────────────────────────────────────────────────────┐
│ 5. 返回最终答案                                          │
│    ├─ 答案内容                                          │
│    ├─ 置信度: 0.85                                      │
│    ├─ 引用来源: [3个政策文档]                           │
│    ├─ 后续建议: ["查看具体补贴金额", "了解申请流程"]   │
│    └─ 响应时间: 1.2秒                                   │
└────────────────┬───────────────────────────────────────┘
                 │
                 ↓
┌────────────────────────────────────────────────────────┐
│ 6. 会话更新                                              │
│    ├─ 保存查询到历史                                    │
│    ├─ 保存答案到历史                                    │
│    ├─ 更新上下文实体                                    │
│    └─ 轮数 + 1                                          │
└────────────────┬───────────────────────────────────────┘
                 │
                 ↓
┌────────────────────────────────────────────────────────┐
│ 7. 监控指标记录                                          │
│    ├─ 记录响应时间: 1.2秒                               │
│    ├─ 记录查询成功                                      │
│    ├─ 记录智能体调用统计                                │
│    └─ 检查告警阈值                                      │
└────────────────────────────────────────────────────────┘
                 │
                 ↓
              返回用户
```
