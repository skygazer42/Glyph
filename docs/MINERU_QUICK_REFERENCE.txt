📁 配置文件
  ├─ .env                          # 环境变量配置
  └─ config/settings.py            # Python 配置类

⚙️ 核心配置项
  ├─ MINERU_ENABLED=true           # 启用 MinerU
  ├─ MINERU_MODE=auto              # 运行模式（官方/本地/自动）
  ├─ MINERU_API_KEY=               # 官方 API 密钥
  └─ MINERU_API_BASE_URL=...       # 本地服务地址

🚀 快速开始
  1. 编辑 .env 文件
  2. python scripts/check_mineru_config.py
  3. python examples/mineru_test.py
  4. 开始使用！

┌─────────────────────────────────────────────────────────────────┐
│                        三种运行模式                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  official  →  使用官方云服务 API (mineru.net)                   │
│               ✓ 无需部署                                        │
│               ✗ 需要 API Key                                    │
│                                                                 │
│  local     →  使用本地部署服务                                  │
│               ✓ 无限制使用                                      │
│               ✗ 需要部署服务                                    │
│                                                                 │
│  auto      →  自动选择（推荐）                                  │
│               ✓ 有 Key 用官方                                   │
│               ✓ 无 Key 用本地                                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                        使用示例                                  │
├─────────────────────────────────────────────────────────────────┤

│                                                                 │

│                                                                 │
