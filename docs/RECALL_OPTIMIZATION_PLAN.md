## 优化策略

### 策略1: 提高相似度阈值 ✓
**问题**: threshold=0.0 导致低相关文档也被召回
**方案**: 设置合理阈值，过滤低分结果

### 策略2: 增强查询（Query Enhancement）✓
**问题**: 简短查询缺乏上下文
**方案**:
- 查询扩展
- 添加关键词
- 同义词替换

### 策略3: 文档分段优化 ✓
**问题**: 长文档整体嵌入，局部信息丢失
**方案**:
- 对长文档进行分段
- 段落级别嵌入
- 提高检索粒度

### 策略4: 添加 Reranker ✓
**问题**: 初步召回准确度不足
**方案**:
- 使用 DashScope Rerank API
- 对初召回结果重新排序
- 提高 Top-K 精度

### 策略5: 混合检索 ✓
**问题**: 纯向量检索可能遗漏关键词匹配
**方案**:
- 结合 BM25 关键词检索
- 融合向量和关键词分数

### 策略6: 元数据增强 ✓
**问题**: 文档分类不够细化
**方案**:
- 添加更多元数据字段
- 关键词提取
- 适用对象标注

## 优先级排序

1. **高优先级（立即实施）**
   - ✅ 提高相似度阈值（简单有效）
   - ✅ 文档分段优化（提高检索粒度）
   - ✅ 添加 Reranker（显著提升精度）

2. **中优先级（后续优化）**
   - 查询增强
   - 混合检索

3. **低优先级（长期优化）**
   - 元数据增强
   - 用户反馈学习
