## 3. Bug修复 ✓

### 修复的问题
在`hierarchical_index.py:364-370`修复了chunk切分的无限循环bug：

**修改前（会导致MemoryError）：**
```python
start = end - self.config.chunk_overlap
if start <= 0:
    start = end  # Bug: 可能导致不向前移动
```

**修改后：**
```python
new_start = end - self.config.chunk_overlap
if new_start <= start:
    start = end  # 确保窗口向前移动
else:
    start = new_start
```
